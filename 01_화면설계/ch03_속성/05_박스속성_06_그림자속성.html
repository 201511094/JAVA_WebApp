5. 박스속성

종류

width - 폭(너비)
height - 높이
padding - 안쪽 여백
border - 테두리
margin - 주변 여백

width, height 속성

크기단위 입력 (px % em vw vh)

padding 속성

padding-top/right/bottom/left
padding: 0 --> 상우하좌 10px
padding: 0 10px; --> 상하0 우좌10px
padding: 0 10px 20px; --> 상0 우좌10px 하20px
padding: 0 10px 20px 30px --> 상0 우10px 하20px 좌30px

margin 속성

padding과 입력방법 같음

border 속성

3가지 속성이 모두 입력되어야 정확히 출력됨
1) border-width - 두께(크기단위 입력)
2) border-color - 색상(색상단위 입력)
3) border-style - 테두리 스타일 주로 한번에 입력, 순서는 상관 없음
입력 방식 - border: width style color --> ex) border: 2px solid #f00;
방향별 border 설정 시 border-top border-right border-bottom border-left

border-style

none - 없음
solid - 실선
dashed - 대시선
dotted - 점선
double - 이중선
outset - 양각선
inset - 음각선
ridge - 테두리만 양각선
groove - 테두리만 음각선

border-radius

박스모델 모서리부분을 둥글게 만들때, 동일한 값일 경우 하나의 값만 입력
border-radius: 좌상 우상 우하 좌하; 왼쪽 상단기준으로 4개 값을 시계방향으로 차례대로 적음
