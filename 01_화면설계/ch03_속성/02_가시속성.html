2. 가시속성

태그가 화면에 보여지는 방식을 지정, display / visibility / opacity
text text
BOX MODEL
text text

display 속성

block: 블럭형식, 공간개념으로 행단위 영역을 차지하며 폭과 높이 설정이 가능
inline-block: 인라인-블럭형식, 이미지같이 폭과 높이를 가지면서 자리는 글자처럼 차지
inline: 인라인형식, text(입력된 만큼만 자리잡고 있음)
none: 화면에 보이지 않게

visibility 속성

visible - 화면에 보이게
hidden - 화면에 보이지 않게

opacity 속성

투명도, 0~1사이의 정수로 표현
