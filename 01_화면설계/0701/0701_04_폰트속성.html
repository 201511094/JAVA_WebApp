폰트 속성

종류

font-family - 서체, 서체, "한글서체", 한글서체의 영문명, 기본서체제공
font-size - 글자 크기, (크기 단위)
font-weight - 두께 normal bold (strong b h1~h6 기본으로 두꺼운 서체 초기화 시 normal 표시)
font-style - 기울기 normal italic (em i address 초기화 시 normal로 표시)
color - 글자색
line-height - 줄간격
letter-spacing - 자간(글자사이)
word-spacing - 단어 간격
text-align - 글자 정렬 left center right justify(양쪽)
text-decoration - 밑줄 none underline (a 초기화 시 none으로 표시)

font속성에 한번에 값을 입력할 경우

font: sytle weight size / line-height family, family, family;
color: 글자색;
