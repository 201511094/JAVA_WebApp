2.선택자

전체선택자 - *

  • HTML 내부의 모든 태그 선택
  • html 태그를 포함하여 head, title, body, h1, h2, ul, li 등...

태그선택자 - 태그명

제목(h2태그)

특정 태그를 선택할 때(p태그)


아이디선택자 - #

태그에 id속성을 가진 선택자
id는 동일한 아이디명을 한 곳에서만 사용 가능
layout 구성 시 주요 영역에 id명을 적용하는 경우가 많음, 특정 강조하고 싶은 영역에 id명을 쓰기도 함

아이디 선택자 적용(id=p1)


클래스선택자 - .

태그에 class속성을 가진 선택자
html문서 어디든 동일한 클래스명 사용 가능, class속성에 여러 개의 클래스명을 적용할 수 있음
콘텐츠박스처럼 내용은 다르지만 디자인이 같을 경우 클래스적용이 가능

클래스 선택자 적용(class=p2 p3)
